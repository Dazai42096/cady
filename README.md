# CADY EST Platform

منصة إدارة مبيعات وصيانة المولدات الكهربائية لمؤسسة كادي (CADY EST) - نظام مؤتمت بالكامل مبني على إطار العمل Laravel مع واجهات تدعم اللغة العربية والتصميم المتناسق للاتجاه من اليمين لليسار (RTL).

---

## الميزات الرئيسية (Key Features)

1. **إدارة العملاء والمولدات (Customers & Generators)**:
   - متابعة وتعديل بيانات العملاء وحالاتهم (نشط، قيد الانتظار، موقوف).
   - جرد المولدات التابعة لكل عميل وأرقامها التسلسلية وقدراتها الكهربائية (KVA).
   
2. **عروض الأسعار وعقود الصيانة (Quotations & Maintenance Contracts)**:
   - إنشاء عروض الأسعار لبنود وقطع الغيار واحتساب الضريبة (15%) تلقائياً.
   - تفعيل عقود صيانة سنوية مع جدولة زيارات دورية تلقائية حسب تكرار الزيارة المطلوب.
   - تصدير عروض الأسعار وعقود الصيانة بصيغة PDF واضحة ومنسقة.

3. **زيارات الصيانة الميدانية (Field Visits)**:
   - تعيين الزيارات للمهندسين وفنيي الصيانة.
   - متابعة سير عمل الزيارة الميدانية (مجدولة -> مؤكدة -> قيد العمل -> مكتملة/ملغاة).
   - توثيق تقارير الفحص الفني وملاحظات العملاء.

4. **بوابة العملاء (Customer Portal)**:
   - لوحة تحكم خاصة بكل عميل لمتابعة مولداته، عقوده، وزياراته القادمة.
   - إمكانية تنزيل عروض الأسعار المعتمدة وعقود الصيانة بصيغة PDF مباشرة.

5. **سجل التدقيق والأمان (Security & Audit Logs)**:
   - توثيق كامل للعمليات الحساسة التي يقوم بها المستخدمون في سجل التدقيق (Audit Logs).
   - صلاحيات وصول دقيقة تعتمد على الأدوار (RBAC) باستخدام Policies وميدل وير مخصص.

## متطلبات التشغيل والتهيئة (Installation & Setup)

1. **نسخ ملف البيئة وتوليد المفتاح**:
   ```bash
   cp .env.example .env
   php artisan key:generate
   ```

2. **تثبيت حزم المعتمديات**:
   ```bash
   composer install
   npm install && npm run build
   ```

3. **تهيئة قاعدة البيانات والتغذية (Migrations & Seeding)**:
   ```bash
   php artisan migrate:fresh --seed
   ```

4. **تشغيل خادم التطوير المباشر**:
   ```bash
   php artisan serve
   ```

5. **تشغيل الاختبارات البرمجية**:
   ```bash
   php artisan test
   ```
